Easy to use Android application for navigation within Silicon Valley using public transportation (VTA). Simple interface, powerful functionality. Metafora makes use of live locations and feeds for lightrails and buses to give users a unique transit experience.

Here2There

Here2There is a light-weight web application for Salesforce.org Metadata migration, backups, and storage. It combines a lightweight AngularJS front-end interface with a powerful Node.js back-end for fast movement of your Salesforce.org Metadata.
